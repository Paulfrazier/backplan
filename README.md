# Backplan

A tiny backwards time planner. Enter a target time, add the steps with durations, and
see when you need to start.

Live at **https://backplan.fairpoint.website** (after deploy).

Single-file static app — `index.html` with inline CSS and vanilla JS. No build step.
Open it in a browser, or deploy by pushing this folder to Vercel as a static site.

Templates are saved to `localStorage` on the device. No accounts, no server.

Part of [Fairpoint](https://fairpoint.website).
